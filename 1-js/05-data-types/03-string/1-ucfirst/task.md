# İlk karakteri büyük harf yapma

`ucFirst(str)` adında bir fonksiyon yazın bu fonksiyon verilen `str` değişkeninin ilk karakterini büyük harf yapıp dönsün, örneğin:

```js
ucFirst("ahmet") == "Ahmet";
```
